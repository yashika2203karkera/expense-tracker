# expense-tracker
💰 Personal Expense Tracker with Insights
📌 Project Overview
The Personal Expense Tracker with Insights is a web-based application developed using Python Flask and SQLite. It helps users track their daily expenses, categorize spending, and analyze financial habits through a simple dashboard.
This project is designed to provide a clear understanding of personal finance management in an easy and interactive way.
🚀 Features
➕ Add daily expenses
📋 View all expense records
📂 Category-wise expense tracking
💰 Total expense calculation
📊 Basic insights dashboard
🎨 Simple and clean UI with CSS
🛠️ Technologies Used
Python
Flask
SQLite
HTML
CSS
